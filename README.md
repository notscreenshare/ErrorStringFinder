## Installation
### **Shell (recommended)**
##### Just run this command in your shell
```sh
curl -sSL https://raw.githubusercontent.com/notscreenshare/ErrorStringFinder/refs/heads/main/download-and-run.sh | bash
```

### **Compile yourself**
##### You can easily compile ESF
```sh
./compile.sh
```

## **Info:**
### How to use?:
##### 1. Open ESF
```sh
sudo -E <esf-file>
```

